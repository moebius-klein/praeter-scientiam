# praeter-scientiam  
**Wissen vor der Grenze**

praeter-scientiam ist ein experimenteller Webraum, der nicht Inhalte präsentiert, sondern Relationen sichtbar macht.  
Eine Anwendung jenseits wissenschaftlicher Kategorien, angesiedelt im Zwischenraum vor der Grenze des Wissens.

---

## ✧ Was ist praeter-scientiam?
Ein Projekt, das Relation statt Objekte zeigt.  
Ein Interface, das keine lineare Navigation besitzt, sondern Prozesse.  
Ein Webraum, der Input transformiert, anstatt Seiten auszuspielen.

Kurz:  
**Eine Exploration des Δ – des Zwischenraums im Zentrum des Siegels.**

---

## ✧ Ziele
- relationales Denken erfahrbar machen  
- Übergänge statt Begriffe zeigen  
- eine Webarchitektur nach dem E-V-A-Prinzip gestalten  
- Codex-Δ-Konzepte in eine interaktive Struktur überführen
